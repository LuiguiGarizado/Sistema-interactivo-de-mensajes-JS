# Interactive User Validation

This project shows a simple interactive JavaScript example that asks the user for their name and age, then validates the input and responds accordingly.

## Main files
- `index1.html` - User interface for the interactive prompt.
- `sistemainteractivo.js` - Main logic for age validation and console messaging.

## Implemented features
- Prompt the user for name and age.
- Convert age input into a numeric value.
- Validate whether the entered age is a number.
- Display different messages depending on whether the user is underage or of legal age.

## JavaScript behavior in `sistemainteractivo.js`
- `prompt()` is used to request user input.
- `Number()` converts the age string to a number.
- Validation checks handle invalid input and age thresholds.
- `console.log()` and `console.error()` display results in the browser console.

## Usage
1. Open `index1.html` in a browser.
2. Enter a name and age when prompted.
3. Observe the validation message in the console.
