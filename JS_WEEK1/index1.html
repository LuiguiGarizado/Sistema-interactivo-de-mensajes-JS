Interactive User Validation

This exercise asks for a name and age, validates the input, and displays results in the browser console.

How it works

  • User prompts Requests user input for name and age using `prompt()`.
  • Input validation Parses the age with `Number()` and checks whether it is valid.
  • Age response Displays a console message based on whether the user is underage or of legal age.
