# File Management and Data Structures

This project demonstrates basic JavaScript data handling using arrays, sets, maps, and validation logic.

## Main files
- `index2.html` - Example HTML file for the project.
- `gestionArchivo.js` - Main JavaScript logic with arrays, sets, maps, and validations.

## Implemented concepts
- Create an array of product objects with `id`, `name`, and `price`.
- Use a `Set` to store unique prices and demonstrate duplicate removal.
- Use a `Map` to associate product categories with product names.
- Iterate through objects with `for...in` and `Map.forEach()`.
- Validate product data before use.

## JavaScript features in `gestionArchivo.js`
- `products` array: contains product objects with unique IDs and prices.
- `prices` set: automatically removes duplicate price values.
- `productMap` map: maps categories to product names.
- `validateProducts(list)`: checks product IDs, names, and prices for validity.

## Usage
1. Open `index2.html` in the browser or run `gestionArchivo.js` in the console.
2. Observe the console output for set iteration, map values, and validation results.
3. Check the final product and price structures printed to the console.
