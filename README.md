# JavaScript Interactive Projects

This repository contains four small JavaScript projects that demonstrate core front-end concepts such as user interaction, DOM manipulation, data structures, local storage persistence, and API communication.

## Included projects

### 1. `JS_WEEK1` — Interactive user validation
- File: `JS_WEEK1/sistemainteractivo.js`
- Description: Prompts the user for name and age, validates numeric input, and prints an age-based message to the console.
- Concepts: `prompt()`, input validation, `Number()`, conditional logic, console output.

### 2. `JS_WEEK2` — Data structures and validations
- Files: `JS_WEEK2/gestionArchivo.js`, `JS_WEEK2/index2.html`
- Description: Creates products using arrays, demonstrates unique values with a `Set`, associates categories with a `Map`, and validates product data.
- Concepts: arrays, `Set`, `Map`, iteration, data validation, console logging.

### 3. `JS__WEEK3` — DOM manipulation and local storage
- Files: `JS__WEEK3/manipulatingDOM.js`, `JS__WEEK3/index3.html`, `JS__WEEK3/style.css`
- Description: Builds a note-taking interface that lets users add and delete notes while saving them in `localStorage`.
- Concepts: DOM creation, event listeners, dynamic rendering, persistence with `localStorage`.

### 4. `JS_WEEK4` — Task persistence and API sync
- Files: `JS_WEEK4/PersistenciaDeDatos.js`, `JS_WEEK4/index4.html`, `JS_WEEK4/styles.css`
- Description: Manages task creation, completion, deletion, and persistence with `localStorage`, while also syncing data with a test JSONPlaceholder API.
- Concepts: DOM updates, local storage, asynchronous API calls, `fetch`, `async/await`, CRUD simulation.

## How to use
1. Open the project folder for the desired week.
2. Launch the corresponding `indexX.html` file in a browser.
3. Use the interface or console prompts to test the functionality.
4. Review the individual project README files for more details.

## Notes
- The repository is structured by weekly exercises.
- Each folder contains a separate demonstration of fundamental JavaScript concepts.
- The global README summarizes the purpose of each folder and the main file(s) included.
